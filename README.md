# 🚀 Agency - Modern Digital Agency Website

A modern, responsive, and professional digital agency website built with HTML, CSS, and JavaScript. This template is designed for creative agencies, freelancers, startups, web developers, and digital marketing companies.

## 🌟 Features

- Modern & Clean UI Design
- Fully Responsive Layout
- Dark Theme
- Hero Section
- Services Section
- Portfolio Showcase
- Team Members Section
- Contact Form
- Footer with Quick Links
- Smooth Scrolling
- SEO Friendly Structure
- Fast Loading Speed
- Cross Browser Compatible
- Easy to Customize
